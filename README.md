ReliefSync AI is a multilingual crisis message analysis system
that extracts priority, location, contact details, and assigns a trust score.

# Features
- Crisis message extraction
- Priority classification (High/Medium/Low)
- Multilingual support
- Trust Score estimation
- CSV export functionality

# Tech Stack
- Python 3.11
- Streamlit
- Google Gemini API
- Pandas

# How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run application:
   streamlit run app.py

#  Project Type
AI-based Emergency Response System
